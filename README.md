# device_modelling-lab

Collection of ngspice simulation codes written as a part of coursework

Includes simulations for:
1. CE amplifier
2. D flip-flop
3. Rectifier
2. 1 bit adder
3. 4 bit adder
4. Op amp
5. Cmos inverter and nand
6. Integrator 
7. Differentiator
8. RC circuit
9. Practical differentiator
10. Voltage divider
11. Subtractor

...and many more.
