# device_modelling-lab

Collection of ngspice simulation codes written as a part of coursework

Includes simulations for:
  1. CE amplifier
  2. D flip-flop
  3. Rectifier
  4. 1 bit adder
  5. 4 bit adder
  6. Op amp
  7. Cmos inverter and nand
  8. Integrator 
  9. Differentiator
  10. RC circuit
  11. Practical differentiator
  12. Voltage divider
  13. Subtractor

...and many more.
